= Nomstrocity
Nomstrocity is a simplified Yelp clone built using Rails 4.1.9 to fulfill part of the {Firehose Project}[https://www.thefirehoseproject.com/] curriculum.

== Concepts/Technologies Covered
* \User authentication
* SendGrid[https://sendgrid.com/]
* {Amazon S3}[https://aws.amazon.com/s3/]
* CarrierWave[https://github.com/carrierwaveuploader/carrierwave]
* Figaro[https://github.com/laserlemon/figaro]
* {Google Maps API}[https://developers.google.com/maps/documentation/javascript/tutorial]
* Geocoder[https://github.com/alexreisner/geocoder]
* Pagination with Kaminari[https://github.com/amatsuda/kaminari]
* Using custom fonts with Rails
